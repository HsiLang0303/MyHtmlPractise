# HTML 学习项目

这是一个用于学习和实践 HTML/CSS 的项目仓库。

## 项目结构

- `xiaomi/`: 小米官网首页练习项目
- `12306/`: 12306 网站练习项目

## 技术栈

- HTML5
- CSS3
- 现代化的响应式布局技术(Flexbox/Grid)

## 项目目标

1. 掌握 HTML5 和 CSS3 的基础知识
2. 学习现代化的网页布局技术
3. 实践真实项目的开发流程
4. 提高网页开发和设计能力

## 开发规范

- 使用语义化的 HTML 标签
- 采用 BEM 命名规范
- 注重代码可读性和维护性
- 保持代码整洁和模块化
